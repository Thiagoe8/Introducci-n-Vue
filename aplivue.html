Aplicación en Vue local (no desde CDN)

Se recomienda instalar Vue Devtools en el navegador para inspeccionar y depurar la app. Para crear un proyecto Vue, usás el comando: npm create vue@latest Durante la creación, podés elegir opciones como: TypeScript JSX Vue Router Pinia (estado) Testing (Vitest / Cypress, etc.) ESLint y Prettier DevTools Una vez creado el proyecto: cd nombre-del-proyecto npm install npm run dev Esto levanta el servidor local y ejecuta la app. El proyecto usa la Composition API por defecto.
